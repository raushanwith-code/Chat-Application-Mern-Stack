# 💬 Real-Time Chat Application (MERN + Socket.io) 🚀

A fully functional **Real-Time Chat Application** built using the **MERN Stack** with **Socket.io** for instant messaging.
This project enables users to chat in real-time with a smooth and responsive UI.

---

## ⚡ Features

* 💬 Real-Time Messaging (Instant chat with Socket.io)
* 🟢 Online / Offline User Status
* 👤 User Authentication (Login / Signup)
* 🧑‍🤝‍🧑 One-to-One Chat System
* 🔔 Live Notifications
* 📡 Persistent Chat Data (MongoDB)
* ⚡ Fast & Responsive UI
* 🎯 Clean Component-Based Architecture

---

## 🛠️ Tech Stack

## 🚀 How to Run Locally

### 🔧 1. Clone the repo

```bash
git clone https://github.com/raushanwith-code/Chat-Application-Mern-Stack.git
```

---

### 📦 2. Setup Backend

```bash
cd backend
npm install
npm start
```

---

### 🎨 3. Setup Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```id="p8o5g9"
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
